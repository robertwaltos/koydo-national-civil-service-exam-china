# koydo-national-civil-service-exam-china

National Civil Service Exam China exam preparation — powered by Koydo.

## Quick start

```bash
npm install
npm run dev
```

## Stack

- **Framework:** Next.js 16.2.0 + React 19
- **Styling:** Tailwind CSS 4
- **Backend:** Supabase (shared)
- **Deploy:** Vercel

## Exam Metadata

| Field | Value |
|-------|-------|
| Exam ID | EXAM069 |
| Name | National Civil Service Exam China |
| Country | China |
| Category | Government/civil service |
| Tier | Tier 2 |
| Wave | Wave 4 |
